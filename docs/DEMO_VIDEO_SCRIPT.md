## Recording Notes

**Do:**
- Show real input, real output
- Use the exact input from the script — it's already in the examples
- Go slow. Typing is boring to watch. Paste, hit enter, show output.
- Let the model run for 15-20 seconds without commentary — builds anticipation
- Show the output scrolling and let it land

**Don't:**
- Don't use a slide deck — show the actual tool
- Don't rush the output reveal
- Don't add music or sound effects
- Don't use the word "seamless" or "powerful"
- Don't do a voiceover that sounds like a SaaS ad

**Equipment:**
- OBS (free) for screen recording
- A Snowball or similar USB condenser mic
- Record at 1080p 30fps minimum
- Export at 1080p for YouTube/Vimeo

**Upload:**
- YouTube unlisted link → into PH video field
- Or upload directly to PH if under 500MB
